멋쟁이 사자처럼 프론트엔드 1주차 과제
어려웠던 점
flex direction row와 column 방향 설정이 헷갈렸다
grid-template-areas 구역 이름을 지정하고 배치하는 방법이 어려웠다
justify-content space-between, space-around, space-evenly 차이가 헷갈렸다
align-items 교차 축 기준 정렬이 언제 필요한지 감이 잘 안 잡혔다
grid item 행과 열을 숫자로 구분할 때 비율이 헷갈렸다
  • Flexbox는 메인축과 교차축을 기준으로 아이템을 정렬하는 레이아웃 시스템이다.
  • justify-content는 메인축 방향으로 아이템을 정렬하는 속성이다. center, space-between, space-around, space-evenly 값을 사용할 수 있다.
  • align-items는 교차축 방향으로 아이템을 정렬하는 속성이다. flex-start, center, flex-end, stretch 값을 사용할 수 있다.
  • row-gap은 row 간격을 의미하고 column-gap은은 column 간격 그리고 gap은 row, column의 간격을 의미한다.
  • grid-template-areas는 그리드에 이름을 부여해서 각 영역을 쉽게 관리하고 배치하는 방법이다. "header header" "sidebar main" 이런 식으로 작성한다.
  • HTML 기본 구조는 html, head, body로 이루어져 있다.
